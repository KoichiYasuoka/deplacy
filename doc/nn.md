# [deplacy](https://koichiyasuoka.github.io/deplacy/) for syntaktisk analyse på nynorsk

## med [Stanza](https://stanfordnlp.github.io/stanza)

```py
>>> import stanza
>>> nlp=stanza.Pipeline("nn")
>>> doc=nlp("For me priser dauden, men me elskar live.")
>>> import deplacy
>>> deplacy.render(doc)

me     PRON  <══╗ ║       nsubj

,      PUNCT <══════╗ ║ ║ punct

.      PUNCT <══════════╝ punct
```
