# [deplacy](https://koichiyasuoka.github.io/deplacy/) para análise sintática

## com [spaCy](https://spacy.io/)

```py
>>> import spacy
>>> nlp=spacy.load("pt_core_news_sm")
>>> doc=nlp("Eu quis amar, mas tive medo.")
>>> import deplacy
>>> deplacy.render(doc)
Eu   PRON  <══╗       nsubj
quis VERB  ═╗═╝═══╗═╗ ROOT
amar VERB  <╝     ║ ║ xcomp
,    PUNCT <════╗ ║ ║ punct
mas  CCONJ <══╗ ║ ║ ║ cc
tive VERB  ═╗═╝═╝<╝ ║ conj
medo NOUN  <╝       ║ obj
.    PUNCT <════════╝ punct
```
