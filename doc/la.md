## per [Trankit](https://github.com/nlp-uoregon/trankit)

```py
>>> import trankit
>>> nlp=trankit.Pipeline("latin")
>>> doc=nlp("Cras amet qui numquam amavit, quique amavit cras amet.")
>>> import deplacy
>>> deplacy.render(doc)
```
