# [deplacy](https://koichiyasuoka.github.io/deplacy/) para análisis sintáctico

## con [Stanza](https://stanfordnlp.github.io/stanza)

```py
>>> import stanza
>>> nlp=stanza.Pipeline("es")
>>> doc=nlp("La pluma es la lengua del alma.")
>>> import deplacy
>>> deplacy.render(doc)
La     DET   <╗             det
pluma  NOUN  ═╝<════════╗   nsubj
es     AUX   <════════╗ ║   cop
la     DET   <══════╗ ║ ║   det

de     ADP   <══╗ ║       ║ case
el     DET   <╗ ║ ║       ║ det
alma   NOUN  ═╝═╝<╝       ║ nmod
.      PUNCT <════════════╝ punct
```
