# [deplacy](https://koichiyasuoka.github.io/deplacy/) za sklandenjsko analizo

## z [CLASSLA](https://github.com/clarinsi/classla)

```py
>>> import classla
>>> nlp=classla.Pipeline("sl")
>>> doc=nlp("Po koroškem po kranjskem že ajda zori.")
>>> import deplacy
>>> deplacy.render(doc)
Po        ADP   <╗         case
koroškem  ADJ   ═╝<════╗   obl
po        ADP   <╗     ║   case
kranjskem ADJ   ═╝<══╗ ║   obl
že        PART  <══╗ ║ ║   advmod
ajda      NOUN  <╗ ║ ║ ║   nsubj
zori      VERB  ═╝═╝═╝═╝═╗ root
.         PUNCT <════════╝ punct
```
