## [Camphr-Udify](https://camphr.readthedocs.io/en/latest/notes/udify.html) арқылы талдау

```py
>>> import spacy
>>> nlp=spacy.load("en_udify")
>>> doc=nlp("Тәннен жан артық еді, тәнді жанға бас ұрғызса керек еді.")

>>> deplacy.render(doc)
```
