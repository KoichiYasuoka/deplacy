## le [Stanza](https://stanfordnlp.github.io/stanza)

```py
>>> import stanza
>>> nlp=stanza.Pipeline("ga")
>>> doc=nlp("Táimid faoi dhraíocht ag ceol na farraige.")
>>> import deplacy
>>> deplacy.render(doc)
```
