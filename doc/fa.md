<h2 dir="rtl"> با <a href="https://stanfordnlp.github.io/stanza">Stanza</a></h2>

```py
>>> import stanza
>>> nlp=stanza.Pipeline("fa")
>>> doc=nlp("به اعتقاد من موسيقي هنر نيست، بلكه متافيزيك است.")
>>> import deplacy
>>> deplacy.render(doc,WordRight=True)

```
